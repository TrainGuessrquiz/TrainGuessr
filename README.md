## システム概要

TrainGuessrは、日本の鉄道路線図から駅名を推測するリアルタイム対戦型Webアプリケーションです。シングルプレイとマルチプレイに対応し、WebSocketを活用したリアルタイム通信により、複数ユーザーが同時に参加できるゲーム体験を提供します。

## 技術スタック

### フロントエンド
- **HTML5/CSS3**: セマンティックなマークアップとレスポンシブデザイン
- **JavaScript (ES6+)**: モジュール化されたクライアントサイド処理
- **Socket.IO Client**: リアルタイム双方向通信
- **Canvas API**: 動的な路線図描画とインタラクション

### バックエンド
- **Node.js**: 非同期I/Oによる高パフォーマンスサーバー
- **Express.js**: RESTful APIとWebサーバー機能
- **Socket.IO**: WebSocketベースのリアルタイム通信
- **JSON**: 路線データの構造化管理

### 開発・運用
- **Git**: バージョン管理
- **npm**: パッケージ管理とビルドプロセス
- **モジュラー設計**: 機能別ファイル分割による保守性向上

## 主要機能と技術的特徴

### リアルタイムマルチプレイ
- Socket.IOによる低遅延WebSocket通信

- イベント駆動アーキテクチャによる状態同期

### ゲームロジック
- タイマー機能付きクイズシステム

### UX/UI設計
- レスポンシブWebデザイン（モバイルファースト）

- リアルタイムフィードバック表示
